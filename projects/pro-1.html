Projet : Site Portfolio

Présentation du projet

Ce projet est un site portfolio développé from scratch en utilisant le framework Astro. L'objectif est de se familiariser avec les technolgies du développement Web par l'apprentissage d'un framework ainsi que de consolider mes compétences en JavaScript, HTML et SCSS.

Voir le projet

Astro est un framework web moderne conçu pour créer des sites ultra-rapides en optimisant les performances. Il utilise une approche appelée "Islands Architecture", qui permet de générer des pages principalement statiques, avec seulement des parties interactives chargées lorsque c'est nécessaire. Cela réduit le JavaScript utilisé et améliore les temps de chargement. Astro est compatible avec des outils comme React, Vue, Svelte, et des styles comme Tailwind ou Sass, offrant une grande flexibilité. Il est parfait pour les blogs, portfolios et sites marketing où la rapidité est essentielle.
